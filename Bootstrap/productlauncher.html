About the project

The purpose of this project was to design a software interface for launching multiple products at the same time. Products can be launched on their own without this tool using shortcuts with parameters of the executable files; but that is not a very user-friendly nor a fast approach. The target platform is Windows PC and can be controlled via mouse and keyboard.



REQUIREMENTS
1. Easy to use interface: easy to use software interface for selecting & launching multiple products
2. Show available products to choose from - minimum of 5 products
3. Select products: users can select multiple products to launch and sort the launch order of products
4. Add products: users can select an executable from their folder on their machine
5. Edit launch parameters: parameters can be set separately for each product
6. Save / load presets: users can save selected products and load previously saved preset of products
PLANNING
There were two features that I really wanted to incorporate in the design: a drag & drop area and a giant launch button. A drag & drop area allows for more executable files to be uploaded in a shorter time span and personally, it feels a bit more modern. I also wanted a very visible and bold button for users to click when ready to launch their products, kind of like a self destruct button; when pressed, something big will happen! (In this case, the launching of products and hopefully nothing destructive).
