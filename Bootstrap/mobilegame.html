About the project

This was a project that I worked on in my free time, in order to gain experience in mobile game development. I used Unity as my platform to create my game and coded it in C#. The artwork was done in a variety of software programs. Since I was creating my very first game, I wanted the concept of it to be rather simple; so I chose to make it a platform game. The game can definitely be improved upon, but seeing how this is my first game I worked on, I'm quite proud of it.

My role

I am the artist along with developing a majority of the front-end and back-end of this project. I had guidance and aid from online users via forums and videos. Having learned new software and a programming language, it was a very rewarding experience and well worth the time I spent working on this project.
THE GAME
Inspired by Super Mario Bros and Wings of Vi, I wanted to create an 8-bit style game that takes elements from both games, specifically: the running platform style from Super Mario Bros and the insane difficulty from Wings of Vi. As the player progresses through each stage, the difficulty increases from easy, medium and hard. There is no time limit, but every moving thing is out to kill the player!
BEHIND THE SCENES
The game was built using Unity and the images were drawn in a variety of art programs (Microsoft Paint, SAI EasyPaint and Paint.NET). The UI is made with a canvas that follows the character.
